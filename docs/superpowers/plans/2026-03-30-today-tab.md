# Today Tab — Cartridge Relocation & Lifecycle Fix

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Move all cartridge rendering from the chat tab to a renamed "Today" tab, fix cartridge results disappearing by delaying server-side cleanup, and add all-player leaderboards to arcade games.

**Architecture:** L3 machine stops cleaning up cartridge actors on completion — they stay alive in their final state until day-end or next same-type spawn. Client gets a new TodayTab with card stack + fullscreen takeover for cartridge interaction. BroadcastBar becomes a compact Activity Status Strip. Chat timeline becomes pure messaging with system messages for cartridge events.

**Tech Stack:** XState v5, React 19, Zustand, Framer Motion, Tailwind CSS, Vitest

**Spec:** `docs/superpowers/specs/2026-03-30-today-tab-design.md`

**Worktree:** `.worktrees/activities-tab` (branch: `feature/activities-tab`)

```markdown
## [ADR-124] Today Tab — Cartridge Relocation & Result Hold
*   **Date:** 2026-03-30
*   **Status:** Accepted
*   **Context:** Two problems: (1) cartridges render inline in the chat timeline, cluttering the group chat on mobile and competing with messages; (2) when a cartridge completes, the server immediately stops the child actor and nulls the ref, so the next SYNC delivers `null` — results never render on the client (GH #113, #72). The schedule tab duplicates information already visible in chat (#6). Overlapping cartridge phases cause label confusion (#83).
*   **Decision:**
    1.  **Rename "Schedule" tab → "Today"** — all cartridge rendering moves here as a card stack.
    2.  **Card states:** Upcoming (client-only, from manifest timeline), Live (CTA → fullscreen takeover), Completed (tap → fullscreen results via cartridge's own results mode).
    3.  **Server result hold:** Remove `cleanupXxxCartridge` from `xstate.done.actor` transitions in L3. Cartridge actors stay alive in their final state. Cleanup triggers: (a) day ends (XState auto-stops children on final state), (b) new same-type cartridge spawns (stop previous first).
    4.  **BroadcastBar → Activity Status Strip:** Replace scrolling marquee with compact pills showing live cartridge status. Bell icon opens notifications; tapping elsewhere navigates to Today tab.
    5.  **Chat timeline:** Remove cartridge entries from `useTimeline()`. Inject lightweight system messages for cartridge events.
    6.  **Fullscreen takeover:** Cartridge components render in a fullscreen overlay (same components as today). Tab bar hidden during takeover. Dismiss via X or swipe down.
    7.  **Arcade leaderboards:** Add all-player ranking to the 9 arcade games that only show individual scores. Use `allPlayerResults` field added to game projection after completion.
*   **Consequences:**
    *   Chat tab becomes a clean messaging surface — no cartridge clutter.
    *   Results persist visually until the day ends — fixes #113, #72.
    *   Addresses or partially resolves: #2, #6, #16, #24, #32, #34, #36, #58, #70, #83, #86, #88, #89, #90.
    *   Snapshot size increases slightly (completed cartridge actors remain serialized) — bounded at 4 actors max per day.
    *   Vivid shell only — Classic and Immersive shells unchanged for now.
```

```typescript
import type { SocialPlayer, DailyManifest } from '@pecking-order/shared-types';
import { VoteEvents } from '@pecking-order/shared-types';

function makeRoster(count: number): Record<string, SocialPlayer> {
  const roster: Record<string, SocialPlayer> = {};

      personaName: `Player ${i}`,
      avatarUrl: '',
      status: 'ALIVE',
      silver: 50,
      gold: 0,
      realUserId: `u${i}`,
    } as SocialPlayer;
  }
  return roster;
}

const BASE_MANIFEST: DailyManifest = {
  dayIndex: 1,
  theme: 'Day 1',
  voteType: 'MAJORITY',
  gameType: 'NONE',
  timeline: [],
};

describe('L3 result hold — cartridge actors stay alive after completion', () => {
  it('voting cartridge ref remains non-null after xstate.done.actor fires', () => {
    const roster = makeRoster(4);
    const actor = createActor(dailySessionMachine, {
      input: { dayIndex: 1, roster, manifest: BASE_MANIFEST },
    });
    actor.start();

    // Trigger voting
    actor.send({ type: 'INTERNAL.OPEN_VOTING', payload: {} } as any);
    let snap = actor.getSnapshot();
    expect(snap.context.activeVotingCartridgeRef).not.toBeNull();

    // All players vote for p1
    for (const pid of Object.keys(roster)) {
      actor.send({ type: VoteEvents.MAJORITY.CAST, senderId: pid, targetId: 'p1' } as any);
    }

    // Close voting — triggers calculateResults → final state → xstate.done.actor
    actor.send({ type: 'INTERNAL.CLOSE_VOTING' } as any);

    snap = actor.getSnapshot();
    // Result hold: the ref should still exist
    expect(snap.context.activeVotingCartridgeRef).not.toBeNull();

    // The cartridge's context should contain results
    const votingSnap = snap.context.activeVotingCartridgeRef!.getSnapshot();
    expect(votingSnap.context.results).toBeDefined();
    expect(votingSnap.context.results.eliminatedId).toBe('p1');

    actor.stop();
  });

  it('voting cartridge is cleaned up when day ends', () => {
    const roster = makeRoster(4);
    const actor = createActor(dailySessionMachine, {
      input: { dayIndex: 1, roster, manifest: BASE_MANIFEST },
    });
    actor.start();

    // Trigger and complete voting
    actor.send({ type: 'INTERNAL.OPEN_VOTING', payload: {} } as any);
    for (const pid of Object.keys(roster)) {
      actor.send({ type: VoteEvents.MAJORITY.CAST, senderId: pid, targetId: 'p1' } as any);
    }
    actor.send({ type: 'INTERNAL.CLOSE_VOTING' } as any);

    // End day — L3 enters 'finishing' final state, XState auto-stops children
    actor.send({ type: 'INTERNAL.END_DAY' } as any);
    const snap = actor.getSnapshot();
    expect(snap.status).toBe('done');
  });
});
```

```typescript
cleanupGameChannels: enqueueActions(({ context, enqueue }: any) => {
  const channels = { ...context.channels };
  let hasGameChannels = false;
  for (const [id, ch] of Object.entries(channels)) {
```

```tsx
function getCartridgeState(cartridge: any, kind: string): 'live' | 'completed' {
  if (!cartridge) return 'live';
  // Check various phase field patterns across cartridge types
  const phase = cartridge.phase;
  if (phase === VotingPhases.REVEAL || phase === 'RESULTS' || phase === 'REVEAL' ||
      phase === 'COMPLETED' || phase === 'SCOREBOARD') {
    return 'completed';
  }
  // Arcade games: check status field
  if (cartridge.status === 'COMPLETED') return 'completed';
  return 'live';
}

function getVotingSummary(cartridge: any): string {
  if (!cartridge?.results) return '';
  const elim = cartridge.results.eliminatedId;
  return elim ? `${elim} eliminated` : 'No elimination';
}

function getGameSummary(cartridge: any, playerId: string | null): string {
  if (!cartridge) return '';
  const silver = cartridge.silverReward ?? 0;
  return silver > 0 ? `+${silver} Silver` : 'Complete';
}

function getPromptSummary(cartridge: any): string {
  if (!cartridge?.results) return '';
  const count = Object.keys(cartridge.responses || cartridge.stances || cartridge.choices || {}).length;
  return count > 0 ? `${count} responded` : '';
}

function getDilemmaSummary(cartridge: any): string {
  if (!cartridge?.results) return '';
  return 'Resolved';
}

interface TodayTabProps {
  onOpenCartridge: (kind: string, cartridge: any) => void;
}

export function TodayTab({ onOpenCartridge }: TodayTabProps) {
  const dayIndex = useGameStore(s => s.dayIndex);
  const manifest = useGameStore(s => s.manifest);
  const activeVotingCartridge = useGameStore(s => s.activeVotingCartridge);
  const activeGameCartridge = useGameStore(s => s.activeGameCartridge);
  const activePromptCartridge = useGameStore(s => s.activePromptCartridge);
  const activeDilemma = useGameStore(s => s.activeDilemma);
  const playerId = useGameStore(s => s.playerId);

  const cards: CardData[] = useMemo(() => {
    const result: CardData[] = [];

    // Active/completed cartridges from SYNC
    if (activeVotingCartridge) {
      const state = getCartridgeState(activeVotingCartridge, 'voting');
      result.push({
        kind: 'voting',
        typeKey: activeVotingCartridge.voteType || 'MAJORITY',
        state,
        summaryLine: state === 'completed' ? getVotingSummary(activeVotingCartridge) : undefined,
        cartridge: activeVotingCartridge,
      });
    }

    if (activeGameCartridge) {
      const state = getCartridgeState(activeGameCartridge, 'game');
      result.push({
        kind: 'game',
        typeKey: activeGameCartridge.gameType || 'UNKNOWN',
        state,
        summaryLine: state === 'completed' ? getGameSummary(activeGameCartridge, playerId) : undefined,
        cartridge: activeGameCartridge,
      });
    }

    if (activePromptCartridge) {
      const state = getCartridgeState(activePromptCartridge, 'prompt');
      result.push({
        kind: 'prompt',
        typeKey: activePromptCartridge.promptType || 'UNKNOWN',
        state,
        summaryLine: state === 'completed' ? getPromptSummary(activePromptCartridge) : undefined,
        cartridge: activePromptCartridge,
      });
    }

    if (activeDilemma) {
      const state = getCartridgeState(activeDilemma, 'dilemma');
      result.push({
        kind: 'dilemma',
        typeKey: activeDilemma.dilemmaType || 'UNKNOWN',
        state,
        summaryLine: state === 'completed' ? getDilemmaSummary(activeDilemma) : undefined,
        cartridge: activeDilemma,
      });
    }

    // Upcoming cartridges from manifest timeline
    const days = manifest?.days || [];
    const currentDay = days[dayIndex - 1];
    if (currentDay?.timeline) {
      const now = Date.now();
      for (const event of currentDay.timeline) {
        const eventTime = new Date(event.time).getTime();
        if (eventTime <= now) continue; // past events — already handled via active cartridges

        let kind: 'voting' | 'game' | 'prompt' | 'dilemma' | null = null;
        let typeKey = '';

        if (event.action === 'OPEN_VOTING' || event.action === 'START_VOTING') {
          // Skip if voting is already active/completed
          if (activeVotingCartridge) continue;
          kind = 'voting';
          typeKey = currentDay.voteType || 'MAJORITY';
        } else if (event.action === 'START_GAME' || event.action === 'START_CARTRIDGE') {
          if (activeGameCartridge) continue;
          kind = 'game';
          typeKey = currentDay.gameType || 'UNKNOWN';
        } else if (event.action === 'START_ACTIVITY') {
          if (activePromptCartridge) continue;
          kind = 'prompt';
          typeKey = currentDay.activityType || (event.payload as any)?.promptType || 'UNKNOWN';
        } else if (event.action === 'START_DILEMMA') {
          if (activeDilemma) continue;
          kind = 'dilemma';
          typeKey = currentDay.dilemmaType || 'UNKNOWN';
        }

        if (kind && typeKey && typeKey !== 'NONE') {
          const diffMs = eventTime - now;
          const hours = Math.floor(diffMs / 3600000);
          const mins = Math.floor((diffMs % 3600000) / 60000);
          const countdown = hours > 0 ? `${hours}h ${mins}m` : `${mins}m`;

          result.push({
            kind,
            typeKey,
            state: 'upcoming',
            scheduledTime: eventTime,
          });
        }
      }
    }

    // Sort: completed first, then live, then upcoming (by scheduled time)
    result.sort((a, b) => {
      const order = { completed: 0, live: 1, upcoming: 2 };
      if (order[a.state] !== order[b.state]) return order[a.state] - order[b.state];
      if (a.state === 'upcoming' && b.state === 'upcoming') {
        return (a.scheduledTime || 0) - (b.scheduledTime || 0);
      }
      return 0;
    });

    return result;
  }, [activeVotingCartridge, activeGameCartridge, activePromptCartridge, activeDilemma, manifest, dayIndex, playerId]);

  return (
    <div style={{
      display: 'flex',
      flexDirection: 'column',
      height: '100%',
      overflow: 'hidden',
    }}>
      {/* Header */}
      <div style={{
        padding: '8px 16px',
        display: 'flex',
        justifyContent: 'space-between',
        alignItems: 'center',
        borderBottom: '1px solid rgba(139,115,85,0.1)',
        flexShrink: 0,
      }}>
        <span style={{
          fontSize: 11, fontWeight: 700,
          color: 'var(--vivid-phase-accent)',
          letterSpacing: '0.05em',
          fontFamily: 'var(--vivid-font-mono)',
          textTransform: 'uppercase',
        }}>
          Day {dayIndex} {'\u2014'} Today
        </span>
        <span style={{
          fontSize: 10,
          color: 'rgba(255,255,255,0.4)',
          fontFamily: 'var(--vivid-font-mono)',
        }}>
          {cards.length} {cards.length === 1 ? 'activity' : 'activities'}
        </span>
      </div>

      {/* Card stack */}
      <div style={{
        padding: 12,
        display: 'flex',
        flexDirection: 'column',
        gap: 10,
        flex: 1,
      }}>
        {cards.length === 0 && (
          <div style={{
            display: 'flex', alignItems: 'center', justifyContent: 'center',
            flex: 1, color: 'rgba(255,255,255,0.3)',
            fontSize: 13, fontFamily: 'var(--vivid-font-mono)',
          }}>
            No activities scheduled today
          </div>
        )}
        {cards.map((card, i) => {
          // Calculate countdown for upcoming/live cards
          let countdown: string | null = null;
          if (card.state === 'upcoming' && card.scheduledTime) {
            const diffMs = card.scheduledTime - Date.now();
            if (diffMs > 0) {
              const hours = Math.floor(diffMs / 3600000);
              const mins = Math.floor((diffMs % 3600000) / 60000);
              countdown = hours > 0 ? `${hours}h ${mins}m` : `${mins}m`;
            }
          }

          return (
            <ActivityCard
              key={`${card.kind}-${card.typeKey}-${card.state}`}
              kind={card.kind}
              typeKey={card.typeKey}
              state={card.state}
              countdown={countdown ?? undefined}
              summaryLine={card.summaryLine}
              onTap={() => onOpenCartridge(card.kind, card.cartridge)}
            />
          );
        })}
      </div>
    </div>
  );
}
```

```tsx
import { useGameStore } from '../../../store/useGameStore';
import { PersonaAvatar } from '../../../components/PersonaAvatar';

interface PlayerResult {
  playerId: string;
  silverReward: number;
  result?: Record<string, number>;
}

interface LeaderboardProps {
  allPlayerResults: PlayerResult[];
  currentPlayerId: string;
}

export function Leaderboard({ allPlayerResults, currentPlayerId }: LeaderboardProps) {
  const roster = useGameStore(s => s.roster);

  if (!allPlayerResults || allPlayerResults.length === 0) return null;

  return (
    <div style={{ padding: '0 4px' }}>
      <p style={{
        fontSize: 9, fontWeight: 700,
        color: 'rgba(255,255,255,0.4)',
        textTransform: 'uppercase',
        letterSpacing: '0.08em',
        marginBottom: 8,
        fontFamily: 'var(--vivid-font-mono)',
        textAlign: 'center',
      }}>
        Leaderboard
      </p>
      <div style={{ display: 'flex', flexDirection: 'column', gap: 4 }}>
        {allPlayerResults.map((entry, rank) => {
          const player = roster[entry.playerId];
          const isMe = entry.playerId === currentPlayerId;

          return (
            <div
              key={entry.playerId}
              style={{
                display: 'flex',
                alignItems: 'center',
                gap: 8,
                padding: '8px 10px',
                borderRadius: 10,
                background: isMe ? 'rgba(196,166,106,0.1)' : 'rgba(139,115,85,0.06)',
                border: isMe ? '1px solid rgba(196,166,106,0.25)' : '1px solid transparent',
              }}
            >
              {/* Rank */}
              <span style={{
                width: 20, textAlign: 'center',
                fontSize: 12, fontWeight: 800,
                color: rank === 0 ? 'var(--vivid-phase-accent)' : 'rgba(255,255,255,0.4)',
                fontFamily: 'var(--vivid-font-mono)',
              }}>
                {rank + 1}
              </span>

              {/* Avatar */}
              <PersonaAvatar
                avatarUrl={player?.avatarUrl}
                personaName={player?.personaName}
                size={28}
              />

              {/* Name */}
              <span style={{
                flex: 1, fontSize: 12, fontWeight: 600,
                color: isMe ? 'var(--vivid-phase-accent)' : 'var(--vivid-text-base)',
                fontFamily: 'var(--vivid-font-display)',
              }}>
                {isMe ? 'You' : (player?.personaName || entry.playerId)}
              </span>

              {/* Silver */}
              <span style={{
                fontSize: 11, fontWeight: 700,
                color: 'var(--vivid-phase-accent)',
                fontFamily: 'var(--vivid-font-mono)',
                background: 'rgba(196,166,106,0.15)',
                borderRadius: 12,
                padding: '2px 8px',
              }}>
                +{entry.silverReward}
              </span>
            </div>
          );
        })}
      </div>
    </div>
  );
}
```
