The Confession Booth

Each day, the CONFESSION channel opens briefly. Alive players drop anonymous confessions under a daily handle (“Confessor #3”). The channel is not a chat thread — it’s a reality-TV interrogation room: single warm sidelight, tape-label handles, a dominant “on air” composer, and a reel of cassette cards where others’ confessions sit framed like evidence. Posts arrive anonymized; the server never tells the client who said what.

Edge: not in the booth

A player with myHandle === null — typically eliminated before the phase opened, or joining the game mid-day — sees a minimal empty state. They can still browse the archived cassettes later, but during the live phase they see only the chrome and a single “locked out” plate where the composer would sit. The reel collapses; there’s nothing to say, so the room doesn’t try to fill itself with activity.

Design notes

Identity = tape, not name

The daily handle is rendered as an evidence tape label (skewed pink strip, Clash 800 · 0.22em), never as body text. Avatars everywhere else in Pulse are photos; here, the mask-silhouette replaces the photo and the tape replaces the name. Two visual substitutions tell the player they’re anonymous without writing a single word.

Room, not thread

A single amber sidelight from top-left + pink undertone from bottom-right reads as “one lamp, one speaker.” The scan-line texture (1px / 3px at 1.2% opacity) adds a VHS tape feel without overstating it. Corner vignettes on cassettes reinforce spotlight-on-the-speaker.

Composer is the set

~40% of viewport is the booth frame, not a 48px input. Placeholder is italic Satoshi (“Speak into the mic…”); live text is Clash 500 · 19px so the draft already reads like the final post. GO ON AIR replaces “Send” — cost-of-action is visible.

One accent per state

Pink = identity (tape). Red = live (ON AIR, REC). Gold = warning (LAST CALL). Closed drops all three to neutral. Only one of these is ever “the loud one” in a given state — follows Pulse’s 10% rule.

Nameplate, not reel

Entry is a one-time ignition beat (~800ms: mask drops, number stamps, caption fades in, auto-continues on first tap). Not an every-day reveal-grade reel — that would overload a recurring event. Re-playable by tapping the tape label in the composer.

Rules always on

One short line under the composer (“Once submitted, cannot be edited or taken back. Everyone sees the tape; no one sees the name.”) — no “?” button, no tooltip. Matches Pulse’s rules-always-visible principle. Copy is the explainer.

Archive lives past closing

Phase close replaces the composer with a rubber-stamp plate; cassettes stay browsable — desaturated slightly (saturate(0.85)), spotlight damped. The booth reads as “closed door to the room, tapes on the shelf outside.”

Non-member ≠ empty state

If myHandle === null, don’t show the reel. A player who wasn’t in the phase shouldn’t see live tapes in real time; they’ll come to the archive at end-of-day. The “locked out” plate respects that and doesn’t leak activity.
