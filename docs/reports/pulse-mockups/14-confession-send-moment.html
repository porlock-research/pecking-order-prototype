Text flies to reel (sender)

View Transitions morphs the element carrying view-transition-name: flying-tape from its position inside the mic-frame to its new home inside a cassette at the top of the reel. Browser does the arc automatically.

t = 0–620 ms

Cassette drops (recipient)

cassette-drop: Y from −70 px → +6 px → 0 px with overshoot. Spring cubic-bezier(.2,1.2,.3,1). Slight blur on entry, sharp on landing.

t = 140 ms

Tape label stamps

tape-stamp: scale 1.18 → 0.96 → 1, 540 ms with bigger overshoot. Feels like it got pressed into place. Tiny Y-drop reinforces the stamp motion.

t = 80–900 ms

Rim-light pulse

cassette-rim animates --rim-strength (0 → 1 → 0) via @property, which interpolates the label's box-shadow-glow. Peak ~280 ms, settled by 900 ms.

t = 600 ms

Mic text cleared. Placeholder fades in. Counter resets to 0 / 280. Send button disabled until next keystroke.

API support & graceful degradation

API Chrome Safari / iOS Firefox Fallback when unsupported
document.startViewTransition()
same-document morph		 111+
Mar 2023		 18.0+
Sep 2024		 134+
Dec 2024		 Feature-detect (if (document.startViewTransition) { … } else { updateDom(); }). Without API, cassette still appears at top of reel via @starting-style entrance — dramatic reveal, just no morph from mic.
@property
rim-light interpolation		 85+ 16.4+ 128+ Rim-light falls back to a fixed-strength glow at steady state. The pulse is lost but the cassette still has a warm frame.
@starting-style
recipient drop-in entry		 117+ 17.5+ 129+ Used as the View-Transition fallback. When unsupported, browsers skip the entry keyframes; new cassette snaps in without the drop. Content is still correct.
prefers-reduced-motion 74+ 10.1+ 63+ All animations collapse to 1 ms (see @media (prefers-reduced-motion) in stylesheet). Send and receive still work; they’re just instant. The peak frame stays legible — that’s the design contract.
CSS springs
via cubic-bezier		 all all all No JS runtime needed; cubic-bezier(.2, 1.2, .3, 1) produces spring-like overshoot natively. Pulse’s existing PULSE_SPRING tokens (Framer Motion) available for complex cases.

The ~3-5% tail of iOS <18 receives the @starting-style-only fallback — the send still works, the cassette still appears, the morph is replaced by a pop-in.
