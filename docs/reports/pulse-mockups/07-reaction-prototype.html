No Long-Press

Every message has a visible ☺ trigger button — subtle on mobile (45% opacity), full on hover/tap. Messages with existing reactions also show a + chip. Both open the same reaction bar. Zero hidden gestures.

Scroll-Safe

The trigger is a tap target, not a gesture. Scrolling never activates it. The ☺ sits inside the message text area (top-right corner) — close enough to tap intentionally, far enough from the scroll axis to avoid accidents.

First-Time Hint

The first message's ☺ button pulses with a subtle coral glow on first visit (CSS animation, 3 cycles). Teaches the interaction without a tooltip or onboarding modal.

Mechanic Actions

Tapping 💰 opens the Send Silver sheet pre-filled with the message author. 💬 navigates to a DM. 👋 sends a nudge immediately. Each has a distinct color (gold, coral, orange) and tiny label for clarity.

Hybrid Social Proof

Actions triggered from a message reaction show as tinted chips ON the message (contextual). Actions from the Cast tab show as broadcast cards in the stream. The ticker shows everything.
