v2.5 changes:
  • Compose button. Pencil icon in the header, with a slot counter pip (e.g. 2/3) showing remaining DM slots today. Tap to enter picking mode. When you're out of slots the icon goes muted and inert. The + chip is removed from the strip.
  • Unified picking flow. Pick 1 → "Start chat with [name]" — same destination as tapping the chip directly, but discoverable for new users. Pick 2+ → "Start group with N". One entry point, both outcomes.
  • Calmer rings. Online / unread / typing / whisper are now static edges (no breathing). Only pending invite pulses — it's the one truly urgent signal that's asking you to act.
  • Prominent typing badge. Coral speech-bubble pill at the chip's bottom-right with 3 animated dots. Replaces the old in-namebar dots.
  • Slot awareness in banner. Picking mode header text shows "(N slots left today)" so you don't get mid-flow surprises.
  • Pending-invite Accept/Decline UI not mocked — in the unified-DM grammar it becomes a state of the DM view: hero renders normally, messages blurred, input row swapped for Accept / Decline buttons. Spec will cover.
  • Cartridge overlays (full-screen cartridge view, pulse bar expansion) deferred to a separate implementation phase after this shell/DM architecture lands.
