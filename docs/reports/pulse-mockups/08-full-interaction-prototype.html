Three Interaction Zones

☺ on messages = emoji reactions + reply (about the message). Avatar tap = player actions (about the player). / commands = proactive actions from input bar. Each concern has its own home.

Progressive Command Builder

Type / or tap a hint chip → command picker (4 cards) → player picker (portrait grid) → amount/confirm → styled preview in input bar → send. Zero typing for game mechanics.

Swipe Right to Reply

Two entry points: ↩ Reply button in the reaction bar (discoverable) + swipe-right shortcut (power users). Both show the reply indicator above the input bar with the quoted message and player color.

Whisper

/whisper transforms the input bar into secret mode — purple tint, lock icon, "Whisper to [Name]..." placeholder. The sent message appears in chat but only the sender and recipient can read it. Others see "🤫 Someone whispered to [Name]".
