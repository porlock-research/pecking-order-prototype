Cold-player onboarding via /harden

The "What to expect" rhythm strip applies four /harden onboarding rules in one element:

Show, don't tell. Three labelled steps + sub-labels. Player parses the daily loop in 2-3 seconds without reading paragraphs.

Smart defaults / progressive disclosure. Top-line essentials only. We could later expose detail behind a "How does this work?" expandable, but cold players don't need it before deciding to tap in.

Don't force long onboarding. The strip is one screen of scroll, not a multi-step intro. Player can ignore it and still complete the form.

Flexible-roster copy

Old: "4 of 8 seats filled" — breaks for DYNAMIC games where the host can keep the room open.

New: "Brenda, Spike, Felix & Max are in. Tap in to join them." — names always work, invitation is universal, no fixed-total assumption. Same surface lights up correctly whether the game is 4-, 6-, 8-, or 12-player.

Server-side, the lobby query (SELECT player_count) can still drive a "seats left" hint when the game is STATIC and the count is meaningful — but it doesn't ship in the headline copy here.

Stereotype corner tags

Each cast card now carries a small red corner ribbon with the persona's stereotype (Sweetheart, Bad Boy, Showmance, Hero). The tag does double duty: it's gossip-energy garnish (you can tell which cast member is which kind of player at a glance) and it places the ONLY non-monochrome element on the card other than the photo.

If the stereotype is too long for the corner (> ~10 chars), truncate or omit. Hosts/personas without a stereotype field skip the tag.

Masthead simplification

Dropped the publishing decoration (Issue №007 · Casting Call). Replaced with a clean two-element bar: wordmark left, code right. The "Casting Call" framing now lives only as the kicker eyebrow above the headline — one place, no redundancy.

Code still rendered in monospace as a "tear-off receipt" — invariant from the old mockup. Useful as a defensive signal in case a player needs to share the code verbally.

Headline accuracy

You're invited. (cast forming) and You're in first. (empty). Both accurate to the actual state — the player has been invited, hasn't joined yet. The accent word ("invited" / "in first") sits in red, carrying the kinetic emphasis the old "in" word did, without claiming a state that isn't true.

CTA copy upgraded: Let's go → → Step in →. Sharper, more theatrical, matches the casting-call kicker.

Voice still loud · gossipy · electric

Loud: 38–52px display headlines, all-caps eyebrows, rhythm-strip step numbers in red.

Gossipy: stereotype tags ("Sweetheart") + cast meta line + Casting Call kicker. The cast is named, the personas are categorised — that's the gossip register.

Electric: red CTA with shadow bevel + glow. Single accent doing all the kinetic work.
